## Программная реализация

1) Построить фазовый портрет системы (векторное поле + основные траектории):

а. Состояния равновесия. Устойчивые состояния равновесия обозначить синими круглыми маркерами, неустойчивые – красными крестообразными маркерами. б. Автоматизировать построение сепаратрис (через собственные векторы матрицы Якоби). Сепаратрисы рисовать пунктирными кривыми красного цвета. в. с. Финитные траектории рисовать зеленым цветом, инфинитные траектории – оранжевым.

2) Построить временные реализации x(t) для характерных траекторий разных типов (состояний равновесия, замкнутых, дважды инфинитных, дважды лимитационных и лимитационно-инфинитных).
